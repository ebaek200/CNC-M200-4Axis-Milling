CNC-M200 Control Panel

ODROID M1S + 7" Touch LCD 기반 독립형 CNC 컨트롤러. CNCjs로 G-code 실행, 조그 제어, 파일 관리를 터치스크린에서 직접 수행.

부품 3개 Armbian + CNCjs
Section 1

System Architecture

컨트롤 패널은 UI/파일 관리만 담당. 실시간 모터 제어는 grblHAL이 전담하는 이중 구조.

┌─────────────────────────────────────────────────────────┐ │ ODROID M1S (Armbian Linux) │ │ ├── CNCjs (Node.js 웹서버) │ │ ├── Chromium Kiosk (전체화면 UI) │ │ └── eMMC 64GB (OS + G-code 파일 저장) │ │ ↕ HDMI + USB Touch │ │ ODROID VU7A Plus 7" LCD ← 터치 조작 + 오디오 알림 │ └────────────────────────┬────────────────────────────────┘ │ USB Serial ▼ ┌─────────────────────────────────────────────────────────┐ │ grblHAL (STM32F446 — FreeRTOS) │ │ ├── 실시간 스텝 펄스 생성 (마이크로초 단위) │ │ ├── 가감속 계산 (Trapezoidal / S-curve) │ │ └── 리미트/비상정지 즉시 반응 │ └────────────────────────┬────────────────────────────────┘ │ STEP / DIR 펄스 ▼ ┌─────────────────────────────────────────────────────────┐ │ DM556 드라이버 ×3 → NEMA23 모터 ×3 (X / Y / Z) │ └─────────────────────────────────────────────────────────┘
핵심 포인트: ODROID(Linux)이 잠깐 느려져도 가공에 영향 없음. 실시간 모터 펄스는 grblHAL(RTOS)이 독립 처리. ODROID은 리모컨 역할.

Section 5

Software Stack

Armbian Linux + CNCjs. 오픈소스 스택으로 커스터마이즈 자유.

OS

Armbian (Ubuntu)

하드커널 공식 이미지. eMMC에 직접 플래시. 경량 서버 모드 (GUI 없음). 메모리 사용 ~120MB.

CNC Controller

CNCjs v2

Node.js 웹서버. 브라우저 기반 UI. grblHAL 프로토콜 지원. G-code 파서 + 3D 비주얼라이저.

Display

Chromium Kiosk

전체화면 키오스크 모드. 부팅 시 자동 실행. localhost:8000 접속. 탭바/주소줄 없음.

Section 6

Workflow

CAM → 파일 전송 → 터치 실행 → 가공 완료. 전 과정 흐름.

1

CAM

PC에서 Fusion 360 / FreeCAD로 G-code (.nc) 생성

2

전송

WiFi로 CNCjs 웹 UI에 드래그&드롭 업로드

3

실행

7" 터치 LCD에서 파일 선택 → START 터치

4

완료

가공 완료 알림음. 파일 영구 보관 → 재사용

파일 전송 방법 (3가지)

방법경로설명
WiFi 업로드 PC 브라우저 → http://odroid-ip:8000 CNCjs 웹 UI에서 드래그&드롭. 가장 편리
USB 메모리 PC → USB → ODROID WiFi 없는 작업장에서 사용
SSH/SCP scp file.nc odroid:/gcode/ 개발자/고급 사용자용
Watch Directory: CNCjs는 /gcode/ 폴더를 자동 감시. 파일이 추가되면 UI에 바로 표시됩니다. eMMC에 영구 저장되므로 전원을 꺼도 파일 유지.

Section 8

Audio Alerts

VU7A Plus의 HDMI 오디오 출력을 활용한 이벤트 알림.

완료

작업 완료 알림
